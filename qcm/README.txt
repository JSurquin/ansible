Utilisation des fichiers QCM :

1. Distribution aux élèves :
   - Donnez le fichier TXT du module en cours
   - Les élèves peuvent le consulter hors ligne
   - Format texte simple, lisible partout

2. Évaluation :
   - Utilisez les QCM pour des quiz rapides
   - 3 questions = ~2-3 minutes par module
   - Réponses incluses pour auto-correction

3. Révision :
   - Les élèves peuvent réviser avec les fichiers
   - Format structuré et clair
   - Explications détaillées
